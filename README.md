# Weather + PM2.5 (Thailand) — README

สรุปงาน
- เว็บ static 3 หน้า:
  - index.html: แผนที่ประเทศไทย สามารถคลิกเลือกจังหวัดเพื่อดูสภาพอากาศเบื้องต้นและค่า PM2.5
  - weather.html: รายการสภาพอากาศแบบยาว 77 จังหวัด พร้อมแถบค้นหา
  - pm25.html: รายการ PM2.5 แบบยาว 77 จังหวัด พร้อมแถบค้นหา

คุณสมบัติเริ่มต้น
- โหมดเริ่มต้น: MOCK (APP_CONFIG.USE_LIVE = false) — ระบบจะสร้างข้อมูลตัวอย่างอัตโนมัติ
- แผนที่ใช้ Leaflet และ GeoJSON จาก public URL (ดูไฟล์ config.js)
- UI เป็นภาษาไทย, โทนสีฟ้าอ่อน, รองรับ responsive

วิธีใช้งาน (local)
1. คลอนหรือดาวน์โหลด repo
2. เปิดไฟล์ด้วย static server เช่น: npx http-server -c-1 .  หรือ serve
3. เปิดเบราว์เซอร์ที่ http://localhost:8080

เชื่อมต่อ API จริง (ตัวเลือก Live)
- แก้ไฟล์ config.js
  - ตั้ง APP_CONFIG.USE_LIVE = true
  - ใส่ APP_CONFIG.WEATHER_API_KEY และ TEMPLATE ให้ตรง
  - กำหนด APP_CONFIG.PM25_API_TEMPLATE เป็น endpoint ที่คืนค่า PM2.5 ระดับจังหวัด

ข้อจำกัดและสิ่งที่ควรปรับปรุง
- ตอนนี้ข้อมูลเป็น mock — หากต้องการเชื่อม API จริง อาจต้องจับคู่พิกัดจังหวัดกับ API ของผู้ให้บริการ
- GeoJSON ถูกโหลดจาก public repo — สามารถย้ายไฟล์ลงใน /data หากต้องการเก็บภายใน repo

ถ้าต้องการ ผมจะ:
- เปลี่ยนโหมดเป็น Live ให้ (ถ้าคุณให้ API key และ endpoint)
- ปรับ UI/สีเพิ่มเติมตามรสนิยม
- เพิ่มการแสดงชื่อจังหวัดบนแผนที่ให้ชัดขึ้น (SVG labels) หรือไอคอนสภาพอากาศ
